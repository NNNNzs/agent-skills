## 常用参数说明

### 创建用户参数

```json
{
  "userName": "testuser",        // 必填，登录用户名
  "nickName": "测试用户",         // 必填，显示昵称
  "password": "123456",          // 必填，密码
  "deptId": 100,                 // 必填，部门 ID
  "phonenumber": "13800138000",  // 可选，手机号
  "email": "test@example.com",   // 可选，邮箱
  "sex": "0",                    // 可选，性别 0男 1女
  "status": "0",                 // 可选，状态 0正常 1停用
  "roleIds": [2]                 // 可选，角色 ID 列表
}
```

### 创建角色参数

```json
{
  "roleName": "测试角色",         // 必填，角色名称
  "roleKey": "test",             // 必填，角色权限字符串
  "roleSort": 10,                // 必填，显示顺序
  "status": "0",                 // 可选，状态 0正常 1停用
  "remark": "角色描述",           // 可选，备注
  "menuIds": [1, 2, 3]           // 可选，菜单 ID 列表
}
```

### 创建菜单参数

```json
{
  "menuName": "系统监控",         // 必填，菜单名称
  "parentId": 0,                 // 必填，父菜单 ID，0 表示根菜单
  "orderNum": 10,                // 必填，显示顺序
  "path": "/monitor",            // 必填，路由地址
  "component": "monitor/index",  // 可选，组件路径
  "menuType": "C",               // 必填，类型 M目录 C菜单 F按钮
  "visible": "0",                // 可选，显示状态 0显示 1隐藏
  "status": "0"                  // 可选，状态 0正常 1停用
}
```

### 创建部门参数

```json
{
  "parentId": 100,               // 必填，父部门 ID，0 表示根部门
  "deptName": "技术部",           // 必填，部门名称
  "orderNum": 10,                // 必填，显示顺序
  "leader": "张三",              // 可选，负责人
  "phone": "13800138000",        // 可选，联系电话
  "email": "tech@example.com"    // 可选，邮箱
}
```
